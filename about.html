About Me and a Brief History

After bouncing around various topics in this phase, I shortly dabbled in C# which inspired me to start looking at other languages beyond just Lua. This led to me learning Python in August of 2020. I've been working with that ever since and have been developing Discord Bots as well as working on various side projects to learn more programming concepts and get more practice in Python.

All of this widened my skillset with general programming concepts and the ideal situations for using them, made programming more natural to do and made me more comfortable doing it as a whole, and lastly it gave me the skills needed to learn just about anything programming related using documentation and practicing it through small projects over time
